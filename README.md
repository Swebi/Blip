# Blip

Blip is an open-source pastebin alternative to easily share code snippets.
Built using NextJS,Typescript, PostgreSQL, Prisma

### Libraries: 
* Clerk
* monaco-editor/react
* shadcn/ui 
* framer-motion
* react-syntax-highlighter
